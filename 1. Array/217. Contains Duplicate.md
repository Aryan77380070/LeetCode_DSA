# 217. Contains Duplicate


## Description

Given an integer array nums, return true if any value appears more than once in the array, otherwise return false.

### Example 1:
- Input: nums = [1, 2, 3, 3]
- Output: true

### Example 2:
- Input: nums = [1, 2, 3, 4]
- Output: false

### Constraints:

- 0 <= nums.length <= 10^5
- -10^9 <= nums[i] <= 10^9

## Python Solution
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        return len(nums) != len(set(nums))
```
### Or
```python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        seen = set()

        for num in nums:
            if num in seen:
                return True
            seen.add(num)

        return False
